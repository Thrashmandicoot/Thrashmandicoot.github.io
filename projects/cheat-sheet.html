Javascript Object

Object Constructor


    function Doge(name, age, furColor){
      this.name = name;
      this.age = age;
      this.furColor = furColor;
      this.bark = function(){
        console.log("wuff");
      };
    }

    var spot = new Doge("Spot", 5, "White");

function doge (name, age, furColor) - sets up constructor with optional parameters that we have set to: name, age, and furColor.

this.parameter - set the objects property equal to the parameter passed through the function.

this.bark = function() - sets bark equal to a function that prints "wuff" to the screen that any doge object that is created can use

Explanation:

This sets up a blueprint for new Doge objects to be created. For example we create a new Doge object named "Spot". We gave him the age of 5 and "White" furColor. These attributes were passed through the constructor function to create our new Doge object with properties as specified. This blueprint is reusable so if we wanted to create other instances of Doge objects we could. Each object is created with its own unique properties but every Doge can bark.

Object Literal


    var fido = {
      name: "Fido",
      age: 5,
      furColor: "White",
      bark: function(){
        console.log("wuff");
      }
    }

This is an example of the same function as used above in the constructor, but as an object literal and we named our object fido. This instantiates a new object named fido, but there is no blueprint provided, which means we cannot instantiate new objects based off of our objects properties.
